基于React+typescript 组件库



安装已经发布的组件库

```javascript
npm install goaljens --save
```

使用

```javascript
// 加载样式
import "goaljens/dist/index.css";
// 引入组件
import { Button } from "goaljens";
```


本地开发命令

```bash
//启动本地环境
npm run stroybook

//跑单元测试
npm test

//build可发布静态文件
npm run build

//发布到 npm
npm run publish
```
